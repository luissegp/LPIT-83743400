Para ejecutar el código parser, se ha de primeramente crear una carpeta llamada "Data" a la misma altura
 de las demás carpetas (Code, Docs,...) que contenga el fichero "cu-lan-ho.log".

Posteriormente ejecutar el código "log-sim.py" (el cuál creará un código del mismo nombre del log que tiene
la carpeta Data, pero que ira copiandose iterativamente), y después una vez ejecutado dicho código ejecutar
el código que procesa la información, es decir, "Parser.py".

Finalmente buscar en internet la URL "http://127.0.0.1:8059/" en la cuál se mostrará las gráficas de cada 
usuario con el tiempo corespondiente en cada segundo, con una ventana deslizante de 10 seg
